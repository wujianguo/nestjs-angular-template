Nestjs + Angular template.

## Features

| Branch              | Feature                            |
| ------------------- | ---------------------------------- |
| main                | Basic server and web client app.   |
| feature/user        | User manager and Authentication.   |
| feature/permission  | Permissions.                       |

## Development

```sh
docker-compose -f docker-compose.dev.yml up
```

## Production

```sh
docker-compose up -d
```
